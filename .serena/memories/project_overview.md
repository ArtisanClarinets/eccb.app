ECCB Management Platform for community band operations: public CMS site, secure member portal, digital music stand, smart upload pipeline, and admin operations. Tech stack: Next.js 16 App Router, React 19, TypeScript, Prisma with MariaDB/MySQL, Better Auth, Redis, BullMQ, Tailwind/Radix/GSAP. Security and compliance are critical (GDPR/HIPAA/PCI-DSS requirements in project guidance).
