## Getting Started

### Accessing the Stand

1. Navigate to the member portal and select "Stand" from the menu
2. Choose an event from your upcoming performances
3. The stand will load with the setlist for that event

### Understanding the Interface

The stand interface consists of:
- **Main Canvas**: Displays the current music page
- **Toolbar**: Contains navigation, annotation, and utility controls
- **Roster Overlay**: Shows which members are currently viewing the piece
- **Page Controls**: Quick navigation between pages and pieces

## Basic Navigation

### Page Navigation

- **Next Page**: Click the right arrow button or swipe right
- **Previous Page**: Click the left arrow button or swipe left
- **Two-Page Mode**: In landscape orientation, pages display side-by-side
- **Half-Page Scroll**: In portrait mode, scroll to view top/bottom halves

### Piece Navigation

- **Next Piece**: Click the "Next Piece" button or use keyboard shortcuts
- **Previous Piece**: Click the "Previous Piece" button or use keyboard shortcuts
- **Setlist View**: Access the full setlist from the toolbar

### Setlist Advance Mode

When enabled, the stand will automatically advance to the next piece when you reach the end of the current piece. This is particularly useful during performances.

## Gig Mode

Gig Mode provides a distraction-free experience optimized for performances:

### Enabling Gig Mode

1. Click the "Gig Mode" toggle in the toolbar
2. The interface will simplify, hiding non-essential controls
3. Navigation becomes gesture-based for quick page turns

### Gig Mode Features

- **Minimal Interface**: Only essential controls are visible
- **Gesture Navigation**: Swipe gestures for page turning
- **Auto-Hide Controls**: Controls fade after inactivity
- **Performance Optimized**: Reduced animations and effects

## Night Mode

Night Mode reduces eye strain in low-light performance environments:

### Enabling Night Mode

1. Click the "Night Mode" toggle in the toolbar
2. The display will switch to a dark color scheme
3. Music notation remains visible with high contrast

### Night Mode Benefits

- **Reduced Eye Strain**: Dark background with light notation
- **Stage-Friendly**: Won't interfere with stage lighting
- **Battery Saving**: Dark themes consume less power on OLED screens

## Navigation Links

Navigation links allow you to create custom shortcuts between different parts of your music:

### Creating Navigation Links

1. Enter Edit Mode from the toolbar
2. Select "Navigation Links" tool
3. Click and drag from your current position to the target location
4. Add a label for the link (e.g., "D.S. al Coda")

### Using Navigation Links

- Click on any active navigation link to jump to its destination
- Links are saved per piece and persist across sessions
- Different layers (Personal, Section, Director) allow for different link sets

## Annotation Tools

The stand supports three annotation layers with different visibility and permissions:

### Annotation Layers

1. **Personal Layer**: Private annotations visible only to you
2. **Section Layer**: Shared within your instrument section
3. **Director Layer**: Created by conductors, visible to all members

### Available Tools

#### Pencil Tool
- **Use Case**: Freehand drawing and marking
- **Features**: Pressure-sensitive line width, customizable color
- **Best For**: Dynamic markings, personal reminders

#### Highlighter Tool
- **Use Case**: Highlighting important passages
- **Features**: Semi-transparent coloring, multiple colors
- **Best For**: Practice reminders, section emphasis

#### Eraser Tool
- **Use Case**: Removing annotations
- **Features**: Size-adjustable eraser, selective layer erasing
- **Best For**: Cleaning up mistakes, updating markings

#### Text Tool
- **Use Case**: Adding text notes
- **Features**: Font size adjustment, color options
- **Best For**: Detailed instructions, reminders

#### Stamp Tool
- **Use Case**: Adding standard musical symbols
- **Features**: Pre-defined symbols (fermata, breath mark, etc.)
- **Best For**: Standard notation additions

### Using Annotation Tools

1. Select your desired tool from the toolbar
2. Choose your color and line width
3. Draw or write on the music canvas
4. Annotations are automatically saved to your selected layer

## Rehearsal Utilities

### Metronome

The built-in metronome helps with practice and ensemble timing:

#### Accessing the Metronome

1. Click the metronome icon in the toolbar
2. The metronome panel will expand

#### Metronome Controls

- **BPM**: Set beats per minute (40-208 range)
- **Time Signature**: Set numerator and denominator (e.g., 4/4, 3/4)
- **Subdivision**: Add eighth or sixteenth note subdivisions
- **Volume**: Adjust click volume
- **Start/Stop**: Begin or pause the metronome

### Tuner

The tuner helps ensure your instrument is properly tuned:

#### Using the Tuner

1. Click the tuner icon in the toolbar
2. Play a note on your instrument
3. The tuner will display your pitch accuracy
4. Adjust your instrument until the indicator is centered

#### Tuner Features

- **Mute Option**: Disable tuner audio for silent tuning
- **Multiple Instruments**: Supports various instrument ranges
- **Visual Feedback**: Clear pitch indication with color coding

### Audio Player

Listen to reference recordings or backing tracks:

#### Loading Audio

1. Click the audio player icon
2. Select from available audio links for the current piece
3. Audio will load and be ready to play

#### Audio Controls

- **Play/Pause**: Start or stop playback
- **Loop Points**: Set custom start and end points for practice
- **Volume**: Adjust playback volume
- **Seek**: Jump to specific points in the audio

### Pitch Pipe

Generate reference pitches for tuning:

#### Using the Pitch Pipe

1. Click the pitch pipe icon
2. Select your desired pitch from the chromatic scale
3. The pitch will sound continuously until stopped

#### Pitch Pipe Options

- **Instrument Types**: Sine, square, triangle, or sawtooth waveforms
- **Pitch Selection**: Full chromatic scale available
- **Volume Control**: Adjustable output level

## Hardware Integration

### Bluetooth Page Turners

Connect wireless page turners for hands-free navigation:

#### Pairing a Page Turner

1. Ensure your page turner is in pairing mode
2. Go to Settings → Hardware → Bluetooth
3. Select your device from the available list
4. Test the connection with the provided buttons

#### Supported Actions

- **Next Page**: Turn to the following page
- **Previous Page**: Return to the previous page
- **Next Piece**: Advance to the next piece in the setlist
- **Previous Piece**: Return to the previous piece

### MIDI Controllers

Connect MIDI devices for advanced control:

#### MIDI Setup

1. Connect your MIDI controller to your device
2. Go to Settings → Hardware → MIDI
3. Map MIDI controls to stand functions
4. Save your configuration

#### Common MIDI Mappings

- **Foot Pedals**: Page turning
- **Expression Pedals**: Volume control
- **Buttons**: Tool switching
- **Knobs**: Zoom and scroll control

## Conductor Features

### Broadcast Commands

Conductors can send commands to all connected members:

#### Available Commands

- **Go to Page**: Jump all members to a specific page
- **Go to Piece**: Navigate to a specific piece in the setlist
- **Clear Annotations**: Remove all annotations from a layer
- **Lock Controls**: Prevent members from changing certain settings

### Roster Management

Monitor which members are actively viewing the music:

#### Roster Features

- **Real-time Presence**: See who's currently viewing each piece
- **Section Filtering**: View presence by instrument section
- **Member Status**: See when members joined or left

## Performance Tips

### Before the Performance

1. **Test Your Setup**: Ensure all hardware is working properly
2. **Charge Devices**: Make sure all devices are fully charged
3. **Check Connectivity**: Verify internet connection if needed
4. **Enable Night Mode**: Reduce screen brightness for stage use

### During Performance

1. **Use Gig Mode**: Minimize distractions during the show
2. **Practice Navigation**: Familiarize yourself with page turning
3. **Monitor Battery**: Keep an eye on device battery levels
4. **Stay Connected**: Ensure your device maintains connection

### Troubleshooting

#### Common Issues

**Page Won't Turn**: Check internet connection, try manual navigation
**Annotations Not Saving**: Verify you're on the correct layer, check permissions
**Audio Not Playing**: Check volume settings, ensure file is loaded
**Hardware Not Responding**: Re-pair Bluetooth/MIDI devices, check batteries

- **Support**: Contact the tech team for assistance
- **Feedback**: Report issues through the member portal
